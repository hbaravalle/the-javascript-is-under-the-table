# Change Theme

Este ejemplo está desarrollado con las herramientas vistas en clase y algunos extras:

- Function Debounce (Es un upgrade que vamos a ver como usar en las siguientes clases)

- Variables
- Query selector y GetElementById
- Objetos literales
- Event listener "Click"
- Condicionales

Como les mencioné anteriormente, estos ejemplos se podrían hacer de una manera mas corta y optimizada. Pero nos pareció interesante que puedan ver que con poquitos elementos aprendidos el único límite es la creatividad. Están invitados a optimizar este ejemplo y puedan resolverlo de una mejor manera.
